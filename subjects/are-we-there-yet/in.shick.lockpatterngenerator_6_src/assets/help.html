Lock Pattern Generator

Program Help

Lock Pattern Generator is a program for creating random patterns with which to lock your Android device.

Please note that the screen lock itself is a built-in part of your device! Lock Pattern Generator cannot lock your device, and it can't unlock it either, it simply suggests patterns for you to use.
This means two important things:

1. Uninstalling Lock Pattern Generator will not turn off the screen lock.
2. If you lock yourself out of your device, the developer can't help you.

Instructions

To start using Lock Pattern Generator, simply press the Generate button. A random pattern will appear on the grid.

If you'd like to try the pattern out before using it, simply press the Practice toggle button. When the Practice toggle is on, the pattern will disappear, and you can try and copy it by swiping on the grid. If you enter the pattern correctly, the nodes will glow blue, and if you are incorrect, they glow red.
While practice mode is on, you can't generate any new patterns. Press the Practice toggle again to return to generator mode and continue creating new patterns.

Once you've selected a pattern to lock your device with, press the Android Security Settings button to leave Lock Pattern Generator and go straight to your device's system settings. There, you can set your pattern to lock your device.
