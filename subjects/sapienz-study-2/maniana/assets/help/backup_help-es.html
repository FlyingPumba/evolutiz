Copias de Seguridad de Maniana

Para hacer Copias de Seguridad de tareas de Maniana, es necesario tener instalado en este disposivo la aplicación gratuita de Dropbox (recomendado) o Google Drive.

Nota: Las últimas restricciones en la aplicación de GMail, provocaron que sea incompatible con las Copias de Seguridad de Maniana.

Hacer una Copia de Seguridad:
Seleccione Copia de Seguridad y a continuación seleccione Dropbox o Google Drive para cargar (subir) el archivo.

Recuperar una Copia de Seguridad:
Abrir el archivo en las aplicaciones de Dropbox o Google Drive y desde allí importar a Maniana.
